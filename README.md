# HTML5 2048 Game

Classic 2048 puzzle game built with HTML5, CSS3, and JavaScript.

## Features
- HTML5 game
- Touch/keyboard controls
- Score tracking
- Responsive design

## Quick Start
```bash
docker build -t html-2048-game .
docker run -p 80:80 html-2048-game
```

Visit: http://localhost
